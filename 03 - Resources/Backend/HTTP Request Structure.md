# HTTP Request Structure

## What
An HTTP request is a message sent from a client to a server.

## Why it matters
> [!note]
## Mental Model
The request is like an order form sent to a restaurant kitchen.

## Example:
It usually contains:  
1. HTTP method  
2. Headers  
3. Body/content  
  
## Structure  
  
### Method  
Defines the action the server should perform.  
  
Examples:  
- GET  
- POST  
- PUT  
- PATCH  
- DELETE  
  
### Headers  
Metadata about the request.  
  
Examples:  
- Authorization  
- Content-Type  
- Accept  
  
### Body  
Optional data sent to the server.  
  
Usually:  
- JSON  
- XML  
- form data  
- files/blobs

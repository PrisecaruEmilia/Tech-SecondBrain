# What is an API

## What
An API allows multiple applications to communicate with each other.

## Why it matters
> [!note]
## Mental Model
Client -> API -> Server

## Example:
- person (client)  
- waitress (API)  
- chef (server)  
  
The client sends a request.  
The server processes it.  
The API facilitates communication.
